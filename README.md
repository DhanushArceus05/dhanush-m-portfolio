# ⚙️ Getting Started

## Clone Repository

```bash
git clone https://github.com/DhanushArceus05/dhanush-m-portfolio.git
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build Production

```bash
npm run build
```
